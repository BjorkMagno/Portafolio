# ⚙️ Back-End Development — APIs & Services (Java + Spring Boot)

Repositorio que reúne proyectos enfocados en el desarrollo de servicios back-end y APIs REST, utilizando Java, Spring Boot y consumo de APIs externas.

El objetivo es demostrar habilidades en:

* Diseño de APIs.
* Seguridad y autenticación.
* Integración con bases de datos.
* Consumo de servicios externos.
* Desarrollo de lógica de negocio estructurada.
  
## 🧠 Proyectos incluidos
### 📌 ForoHub

API REST para gestión de un foro con operaciones CRUD sobre tópicos.
Incluye autenticación con JWT, control de acceso y arquitectura por capas (controller, service, repository, security).

### 📌 LiterAlura

Servicio REST que consume una API externa de libros (Gutendex).
Implementa búsquedas, filtrado, persistencia local y consumo de APIs utilizando HttpClient y Spring Data JPA.

### 📌 muntconverter

Aplicación en Java para conversión de divisas mediante consumo de API externa.
Enfocado en manejo de solicitudes HTTP, procesamiento de JSON y lógica básica de servicios.
