## 🐍 Python Data Analysis & Visualization — Portfolio

Repositorio enfocado en proyectos de análisis y tratamiento de datos en Python, utilizando herramientas de visualización y procesamiento para explorar, limpiar y comprender información.

Los proyectos de modelado más avanzado y desafíos de Machine Learning se encuentran en una carpeta independiente: **desafios**.

## 💡 Enfoque

Este repositorio se centra en:

* Tratamiento y limpieza de datos.
* Análisis exploratorio (EDA).
* Visualización de datos.
* Preparación de datos para modelado.

Se prioriza la comprensión estructural de los datos antes de modelar, manteniendo un enfoque claro y reproducible.

## 🛠️ Herramientas y librerías
* Python (Google Colab / Jupyter)
* pandas — manipulación de datos
* numpy — cálculo numérico
* matplotlib — visualización base
* seaborn — visualización estadística
* scikit-learn

## 📊 Contenido de los proyectos
🔹 Tratamiento de datos

  - Limpieza de datos. (valores nulos, duplicados)
  - Transformaciones y normalización.
  - Conversión de tipos de datos.
  - Preparación para análisis o modelado.
    
🔹 Datos numéricos

  - Estadística descriptiva.
  - Distribuciones.
  - Detección de outliers.
  - Análisis de tendencias.

🔹 Datos categóricos

  - Codificación de variables.
  - Análisis de frecuencias.
  - Comparaciones entre categorías.

🔹 Visualización

  - Gráficas con matplotlib y seaborn.
  - Histogramas, boxplots, scatter plots.
  - Visualización de relaciones entre variables.
  - Representación clara de patrones.

🔹 Realización de demandas de ML y DEV para su modelaje.

Los proyectos siguen generalmente:

* Desarrollar análisis claros y reproducibles.
  
## 📌 Notas
