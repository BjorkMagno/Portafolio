# Dashboard Financiero – Tienda General
## Descripción
Este proyecto consiste en un dashboard financiero desarrollado en Power BI para analizar el desempeño comercial y financiero de una tienda general. El tablero permite visualizar indicadores clave relacionados con ventas, costos, utilidad, descuentos y rendimiento de los equipos de venta. El objetivo principal fue transformar datos operativos en información visual y estratégica para apoyar la toma de decisiones financieras.

## Información que muestra el dashboard
- Ventas totales
- Cantidad total de productos vendidos
- Margen de utilidad
- Promedio de descuentos aplicados
- Comparación entre ventas y costos
- Rendimiento por equipo de ventas
- Indicadores financieros generales

Las visualizaciones permiten identificar tendencias, áreas con mayor rentabilidad y posibles oportunidades de mejora. Gracias al análisis realizado, el dashboard facilita:

- Detectar productos o áreas con menor margen de utilidad.
- Evaluar el impacto de los descuentos sobre las ganancias.
- Comparar ventas contra costos para medir rentabilidad.
- Identificar el desempeño de los equipos de venta.
- Apoyar decisiones relacionadas con estrategias comerciales y control financiero.

## Estructura del modelo en Power BI

El modelo fue estructurado utilizando tablas relacionadas para facilitar el análisis financiero. Se trabajó principalmente con:

- Tabla de ventas
- Tabla de vendedores/equipos de venta
- Medidas financieras en DAX
- Relaciones entre dimensiones y hechos

Además, se diseñaron tarjetas KPI, gráficos comparativos y visualizaciones dinámicas para mejorar la interpretación de la información.

## Tratamiento de datos

Antes del desarrollo del dashboard, se realizó un proceso de tratamiento y limpieza de datos que incluyó:

- Eliminación de registros duplicados
- Corrección de valores nulos o inconsistentes
- Estandarización de formatos
- Validación de información financiera
- Preparación de la base de datos para análisis en Power BI
