# 🚀 Portfolio — Data Science, Back-End & Front-End

Portafolio orientado al desarrollo de soluciones en análisis de datos, APIs backend y aplicaciones frontend, integrando múltiples tecnologías para resolver problemas de forma estructurada y reproducible.

## 💡 Perfil

Experiencia práctica en:

1. Análisis y transformación de datos.
1. Desarrollo de modelos predictivos.
1. Construcción de APIs REST seguras.
1. Consumo e integración de servicios externos.
1. Desarrollo de interfaces interactivas.

Con enfoque en claridad, lógica estructurada y aplicación real de herramientas.

## 🧠 Proyectos clave
### 📊 Data Analysis & Power BI
Desarrollo de dashboards interactivos, modelado de datos y procesos ETL para generación de insights.
### 🐍 Python & Data Processing
Análisis exploratorio, limpieza de datos y visualización para comprensión estructural de datasets.
### 🧠 Machine Learning — TelecomX
Pipeline completo: tratamiento de datos, modelado predictivo, evaluación y exportación de modelos.
### ⚙️ ForoHub API
API REST con autenticación JWT, operaciones CRUD y arquitectura por capas.
### 📚 Gutendex API
Integración con API externa, filtrado de datos y persistencia en base de datos.
### 💱 Currency Converter
Consumo de API externa y procesamiento de datos en Java.
### 🎁 Frontend App (Amigo Secreto)
Aplicación web interactiva con lógica en JavaScript y manipulación del DOM.
